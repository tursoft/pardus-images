## pardus-openjdk

Pardus Base Image üzerinde openjdk jre ve jdk image'lerini içerir.

![Docker Pulls](https://img.shields.io/docker/pulls/tursoft/pardus-openjdk.svg)

### Kullanım
```sh
sudo docker run tursoft/pardus-openjdk:8u181-jre java -version
```
